使用 Web Audio API（無需外部音檔或 npm 套件）為遊戲加入音效回饋，提升遊戲感與沉浸感，並提供靜音開關。

## Acceptance Criteria

- 答對播放輕快上升音，答錯播放低沉音

- 告白場景進場時自動播放夢幻和弦
- 靜音 toggle 可用，狀態跨頁面保留（localStorage）
